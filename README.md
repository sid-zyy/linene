Linene

A minimal and clean full-stack note-taking application.

Linene is built with React, Express, and MongoDB, focusing on simplicity, speed, and an easy-to-understand codebase. It provides a straightforward UI for creating and viewing notes, with built-in rate limiting to prevent spam.
